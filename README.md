# OpenWISP WiFi Login Pages - GSoC Prototype

This repository contains a working prototype demonstrating the architectural improvements proposed for the OpenWISP WiFi Login Pages modernization project as part of Google Summer of Code 2026.

## What's Demonstrated

- **Status Component Refactoring**: Interactive before/after comparison of monolithic vs. modular architecture
- **Captive Portal API**: RFC 8908 implementation with configurable timeouts and fallbacks  
- **React 19 Migration**: Modern hooks, concurrent features, and testing framework updates
- **UI Improvements**: Unified responsive components and performance optimizations

## Key Features

- Interactive demonstrations with visual comparisons
- Working code examples for all proposed changes
- Side-by-side architecture analysis
- Comprehensive test coverage examples
- Setup instructions for local development

## Technologies

- React 19 (with migration examples)
- React Testing Library (replacing Enzyme)
- Modern JavaScript (ES6+)
- Responsive CSS and component architecture
